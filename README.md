# FiapApp 📱

A modern Android application developed for FIAP, showcasing declarative UI design and robust local data management. 

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Local Storage:** Jetpack DataStore (Preferences / Proto)

## ✨ Features
* **Modern UI:** Built entirely with Jetpack Compose for a smooth, reactive user experience.
* **Data Persistence:** Utilizes Jetpack DataStore for asynchronous, consistent, and transactional local data storage, replacing legacy Shared Preferences.

## 🚀 Getting Started
To run this project locally:

1. Clone this repository: `git clone https://github.com/guidonunes/FiapApp.git`
2. Open the project in Android Studio.
3. Sync Gradle to pull down the Compose and DataStore dependencies.
4. Build and run the app on an emulator or physical device.
